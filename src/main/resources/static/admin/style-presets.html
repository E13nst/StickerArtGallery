Добавить пресет

PNG, JPEG или WebP до 3 MB. Отображается как карточка в miniapp.

Файл однажды загружается и сохраняется на сервере как часть глобального пресета. Он используется при каждой генерации пользователем, если в шаблоне есть плейсхолдер {{preset_ref}} (поле автоматически появится в контракте).

Не путать с «фото на одну генерацию» из блока ниже доп. полей: то задаётся в fields типа reference (пример ключа для шаблона — user_face). Форматы: PNG/JPEG/WebP до 3 MB. В API — presetReference*, id img_sagref_*.

Создавай категории в таблице выше. Порядок пресетов — внутри выбранной категории.

Используй {{ключ}} чтобы вставить значение поля в нужное место.

{{prompt}} — куда встраивается свободный текст пользователя  ·  {{emotion}} — поле из списка ниже.

Если в шаблоне нет плейсхолдеров — он добавляется в конец к тексту пользователя.

Лимит для miniapp; сумма maxImages по полям type=reference не должна превышать это значение.

Дополнительные поля

Ключ = плейсхолдер {{emotion}}. Тип reference — слот референсного фото; в промпт подставляется текст Image {index} (или свой шаблон).
