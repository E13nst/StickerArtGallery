Шаблоны формы «Создать свой пресет»

Miniapp: GET /api/generation/user-preset-creation-blueprints. Тело для создания персонального пресета — это настройки по умолчанию (в API поле presetDefaults), смешанные с кодом/именем пользователя → POST /api/generation/style-presets. Плейсхолдер {{preset_ref}} в тексте не задаётся строкой в fields: после создания пресета пользователь загружает опорное фото — PUT /api/style-presets/{id}/reference.

Блок presetDefaults — частичный запрос создания пресета; при сохранении бэкенд проверяет контракт. Слот preset_ref не перечисляется в fields: это опорное фото на сервере после PUT …/reference.

Сюда входят promptSuffix, uiMode, promptInput, fields и т.д. Пользовательские ключи вроде user_face перечисляйте в fields.
