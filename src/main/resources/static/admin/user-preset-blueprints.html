Шаблоны формы «Создать свой пресет»

Здесь задаётся как будет выглядеть форма в miniapp перед созданием личного пресета (GET /api/generation/user-preset-creation-blueprints). Настройки ниже попадают в presetDefaults и тем же контрактом проверяются, что и глобальные пресеты. Плейсхолдер {{preset_ref}} означает опорное фото пресета после POST …/style-presets и загрузки PUT …/{id}/reference.

ID Код Название (админ) Вкл Порядок

Шаблон

{{ключ}} — значение поля. {{prompt}} — свободный текст, если включён блок ниже.

{{preset_ref}} — опорное фото пресета после создания пользователем; в списке полей не задаётся.

Лимит для miniapp; сумма maxImages по полям reference не должна превышать это значение.

Дополнительные поля

Ключ = плейсхолдер {{emotion}}. Тип reference — фото пользователя на одну генерацию.
