  “для общения”, “для любых чатов”, “для друзей”, “на каждый день”, etc.

- The result should look like a real search query or a compressed sequence of search keywords.

  “офисный юмор дедлайны токсичный тимлид митинги zoom”
  “crypto трейдер мемы падение курса panic sell”

OUTPUT FORMAT:
You MUST return STRICT JSON ONLY. No explanations, no markdown, no extra text.

Use EXACTLY this format:
{
  "ru": "search-style description in Russian",
  "en": "search-style description in English"
}

Length requirements:
- Each field must be between 20 and 120 characters.
- Both versions must be semantically equivalent, not direct word-for-word translations.
